MES COMPTES — VERSION PWA IPHONE

Ce dossier contient ta version installable sur iPhone.

Fichiers ajoutés :
- manifest.webmanifest
- service-worker.js
- icons/
- favicon.png

Installation sur iPhone :
1. Mets ce dossier en ligne sur un hébergement gratuit HTTPS (Netlify, Cloudflare Pages ou GitHub Pages).
2. Ouvre l'adresse du site dans Safari sur ton iPhone.
3. Appuie sur Partager.
4. Choisis “Sur l’écran d’accueil”.
5. L’application “Mes Comptes” apparaît avec ton logo.

Important :
- L'installation PWA sur iPhone nécessite une adresse HTTPS.
- Une fois ouverte une première fois, l'app fonctionne hors ligne grâce au cache.
- Les données restent sauvegardées localement dans le navigateur/appareil.
